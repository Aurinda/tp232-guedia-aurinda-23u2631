# Mon Appli d'Info 232

Application web de gestion et d'analyse de données de santé

## Fonctionnalités

- **Enregistrement** de patients (nom, prénom, sexe, âge, activité sportive, maladie)
- **Modification** et suppression des fiches patients
- **Recherche** par nom ou prénom
- **Tableau de bord** avec statistiques en temps réel
- **Analyses statistiques** : moyenne, médiane, écart-type, étendue, distributions
- **Stockage local** (localStorage) — fonctionne sans serveur ni connexion internet


## Utilisation

| Section | Description |
|---|---|
| 🏠 Accueil | Statistiques globales et graphiques |
| ➕ Enregistrer | Ajouter un nouveau patient |
| 👥 Patients | Liste, recherche, modification, suppression |
| 📊 Analyses | Statistiques détaillées (moyenne, médiane, écart-type...) |

## Notes

- Les données sont sauvegardées dans le navigateur (localStorage)
- Chaque utilisateur a ses propres données locales
- Aucune connexion internet requise après chargement de la page
